# OSOP – Arama ve Keşif

Bu proje, Açık Kaynak İşletim Sistemleri dersi kapsamında
Linux paket yönetim sistemleri (apt, snap, flatpak) kullanılarak
paket arama ve keşif işlemlerini gerçekleştirmek için geliştirilmiştir.

## Proje Amacı
- apt search çıktılarının ayrıştırılması
- snap ve flatpak marketlerinde paket araması
- paket detaylarının (versiyon, boyut, bağımlılıklar) gösterilmesi

## Kullanılan Teknolojiler
- Ubuntu Linux
- Python 3
- apt, snap, flatpak

## Proje Yapısı

arastirmalar/  -> literatür ve teknik araştırma
ozellikler/    -> proje gereksinimleri
kaynak/        -> yardımcı notlar
belgeler/      -> ekran görüntüleri ve raporlar
src/           -> kaynak kodlar
